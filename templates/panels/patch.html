자동화 취약점 패치 — Ansible -
운영 중 자동매매 서버 — 안전 우선
미적용 보안 업데이트를 스캔하고 Ansible 플레이북으로 패치를 자동화합니다. 스캔은 읽기 전용(안전)이지만, 실제 패치는 자동매매 봇을 중단시킬 수 있어 자동 실행하지 않습니다. 기본은 dry-run(--check)으로 무엇이 바뀌는지만 확인하고, 실제 적용은 점검 시간대에 수동으로. 실제 적용을 켜려면 .env PATCH_APPLY_ENABLED=True + ansible + passwordless sudo 필요.
