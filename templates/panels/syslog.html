Syslog 수신 — 원격 침해시도 수집
이 패널은 어떻게 봐야 할까?
자동매매 KR / USA 서버가 접속 시도·보안 이벤트를 syslog(UDP/TCP)로 전송하면 여기서 실시간 수신합니다. 파일 tail 방식과 달리 로그 위치가 바뀌어도 끊기지 않습니다. 인증 실패·프로브·SQLi/XSS·포트 스캔 등 의심 이벤트는 자동 분류되어 위협 탐지 → AI 트리아지 → SOAR 파이프라인, 공격 지도, MITRE ATT&CK 에 반영됩니다.
KR/USA 프로젝트에서 이렇게 연결하세요
각 프로젝트에 아래를 추가하면 접속 로그가 이 대시보드로 전송됩니다 (<bind>는 위 수신 주소). 
import logging, logging.handlers
h = logging.handlers.SysLogHandler(address=("127.0.0.1", 5514),
                                   socktype=socket.SOCK_DGRAM)   # 또는 SOCK_STREAM(TCP)
logging.getLogger("werkzeug").addHandler(h)   # 접속 로그를 SOC 로 전송
